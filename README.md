## Development Installation

If Composer is installed globally, run

```bash
composer install
```

View development server:

```bash
bin/cake server -p 8765
```

Then visit `http://localhost:8765` to see the welcome page.

## Testing etc

- Test
```bash
composer test
```

- Code Standards Check
```bash
composer cs-check
```
